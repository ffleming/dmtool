# DMTool

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dmtool'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dmtool

## Usage

Use the classes as you see fit, or use the command-line tools.
